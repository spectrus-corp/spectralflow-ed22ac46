# SpectralFlow

Cyberpunk social platform with:
- Realtime chat
- WebRTC audio/video calls
- Live streaming
- Firebase push notifications
- Supabase backend
- GDPR/legal pages

## Stack
- React + TypeScript + Vite
- Supabase
- Firebase Cloud Messaging
- TailwindCSS
- WebRTC
